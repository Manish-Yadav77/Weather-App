<h2>🛠️ Installation Steps:</h2>

<p>1. Install vite</p>

```
npm create vite@latest
```

<p>2. Install Node js</p>

```
npm i
```

<p>3. Install Tailwind css</p>

```
check website for tailwind
```

<p>4. now run the program</p>

```
npm run dev
```
